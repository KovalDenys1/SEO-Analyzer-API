# SEO Analyzer API

FastAPI service that analyzes a webpage and returns metadata, headings, keyword density, load time, and an SEO score.

Available on [RapidAPI](https://rapidapi.com/KovalDenys1/api/ultimate-seo-analyzer).

## Usage

```http
GET /analyze?url=https://example.com
```
